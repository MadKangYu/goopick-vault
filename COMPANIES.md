# 법인 구분

## 구조

```
goopick-vault/
├── goopick/           # 구픽 (메인)
├── dojangminjok/      # 도장민족
├── madstamp/          # 매드스탬프 (개인회사)
└── madstampinc/       # 매드스탬프 (법인회사)
```
